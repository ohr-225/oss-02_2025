# 오픈소스SW개론 과제 #2

## 1. top
### 개요
- top 명령어는 현재 시스템의 상태를 실시간으로 나타내는 명령어임.
- 실행 중인 프로세스 목록, CPU 사용량

### 주요 옵션
| 옵션 | 설명 |
|------|------|
| -d | 업데이트 간격 변경 |
| -p | 특정 PID만 표시 |
| -u | 특정 사용자 기준 필터 |

### 사용 예시
코드 -> 결과 몇 개 정도 (옵션들 포함)
